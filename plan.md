wir erstellen hier ein plugin für wordpress. das plugin soll eine sponsorenverwaltung darstellen.

aufgaben:
- verwalten von sponsoren (name, link, priority, status=active,draft,etc.)
- assignment von sponsorenlogos
- widget 1: darstellung der logos in einer sidebar, drei logos nebeneinander. reihenfolge über logo prio, optional zufällige anordnung.
- widget 2: scrollen aller logos in einer sidebar von unten nach oben. optional: zufällige reihenfolge der logos beim initialen laden.
